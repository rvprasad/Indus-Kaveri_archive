Appendix - Creating Scope Specifications

Kaveri allows classes, methods and class fields to be entered into the scope. To add these elements to the scope, select the required Class, Method or Field from the Outline View of the Java text editor and choose Indus -> Add to Scope from the context menu. For the Method and Field Specifications, the user is required to enter the name for the scope specification. In case of a class, the user can enter the scope name, change the class name specification to a regular expression matching several classes and pick the scope around the specified type, as indicated earlier. Upon choosing the scope name and the parameters, the scope specification for the chosen element is available to be included in the slice via the Slice the Program dialog box.
